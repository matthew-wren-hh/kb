Band Notes

This repo hosts a minimal band-notes page that is updated manually from weekly PDF emails.

Quick start

  1. Add the newest email PDF to notes/ as latest.pdf.
  2. Update index.html with the latest summary and links.
  3. Open index.html in your browser.

File layout

  • The main page lives at index.html.
  • CSS lives in assets/css/ with component files under assets/css/components/.
  • PDFs and calendar files live in notes/.

Conventions

  • Keep the design minimal and accessible.
  • Use CSS Grid over Flexbox when possible.
  • Avoid external frameworks or dependencies.

Notes

  • Optional: add notes/events.ics to enable the calendar download button.
  • Agent and style preferences live in AGENTS.md.
